#+TITLE:vim-notes

* NAME

vim-notes - Notes about Vim

* Vim-fugitive

- Pressing =o= with the cursor on a commit opens =git show= of that commit in
  a split (instead of pressing Enter which goes to that commit in the same
  pane)

* Commands to remember

Opening file from path in text file

| gf      | open in the same window ("goto file") |
| <c-w>f  | open in a new window (Ctrl-w f) |
| <c-w>gf | open in a new tab (Ctrl-w gf) |

Manpages

| K | Open manpage for string under cursor |

Numeric argument controls manpage section so 7K does =man 7 <str>=
