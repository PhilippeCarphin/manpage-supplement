#+TITLE:man-add

* NAME

man-add - Add a manpage to manpage-supplement-personal repo

* SYNOPSIS

#+begin_src shell
man-add SECTION NAME
#+end_src

* DESCRIPTION

Add a manpage in section number ~SECTION~ with name ~NAME~.

* NOTES

The command verifies that this repo does not already contain a manpage with this
name.
